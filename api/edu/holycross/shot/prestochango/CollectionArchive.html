          Writes a complete RDF description of a Collection Archive in TTL format, without RDF prefix definitions, to a local file. This can be useful when a system like dse that already has prefix statements wants to generate a larger composite RDF description concatenating RDF from multiple sources.

          throws:

          outFile - A local file where output is written.
